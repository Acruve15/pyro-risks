NOAA NCEI module¶

class pyro_risks.datasets.weather.NOAAWeather(source_path: Optional[str] = None, use_cols: Optional[List[str]] = None)[source]¶

Bases: pandas.core.frame.DataFrame

Weather dataset on French territory, accessible upon request to NOAA. Requests are to be made at: https://www.ncdc.noaa.gov/cdo-web.

Parameters

  • source_path – path or URL to your version of the source data

  • use_cols – columns to read from source

kept_cols = ['STATION', 'DATE', 'LATITUDE', 'LONGITUDE', 'ELEVATION', 'DEWP', 'DEWP_ATTRIBUTES', 'FRSHTT', 'GUST', 'MAX', 'MIN', 'MXSPD', 'PRCP', 'SLP', 'SLP_ATTRIBUTES', 'SNDP', 'STP', 'STP_ATTRIBUTES', 'TEMP', 'TEMP_ATTRIBUTES', 'VISIB', 'VISIB_ATTRIBUTES', 'WDSP', 'WDSP_ATTRIBUTES']¶
